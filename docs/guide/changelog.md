# Changelog

All notable changes to MyLib are documented here.

## v1.0.0

Initial stable release.

- `formatDate` with full token support
- `parseDate` with strict and loose modes
- Global configuration via `configure`
- Locale-aware month and weekday names
- Full TypeScript types

## v0.9.0

- Added `parseDate`
- Added locale support to `formatDate`

## v0.8.0

- First public beta
- `formatDate` with basic tokens
